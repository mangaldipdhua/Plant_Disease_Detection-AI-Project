About This Project

Project Overview

This demo project allows users to detect plant diseases by uploading an image. Using a deep learning model, the system predicts diseases, enabling farmers and plant enthusiasts to protect their crops. The website provides a user-friendly interface that guides users through the process, from uploading an image to receiving results with disease information and recommendations.

Model Details

The model is based on a Convolutional Neural Network (CNN) architecture trained to identify various plant diseases. It achieves an accuracy of approximately 92% and a validation loss of 0.15. The model is optimized for quick predictions without compromising accuracy. It classifies diseases in multiple plant species, delivering precise results in just a few seconds.

Dataset Information

The model is trained on the 'New Plant Diseases Dataset' from Kaggle, which contains over 20,000 images across multiple disease categories and healthy plant samples. The dataset includes images of various crops such as tomatoes, potatoes, and corn, each labeled with disease types like early blight, leaf spot, and powdery mildew. The dataset is curated to cover a wide range of plant species and diseases, ensuring broad applicability. The diversity and size of the dataset help improve the robustness of the model.

Model Features

  • Quick real-time prediction for uploaded images
  • Support for a wide range of plant species and diseases
  • High accuracy (around 92%) and low validation loss
  • Detailed disease information with treatment recommendations
  • Scalable and optimized for deployment in agricultural systems
  • User-friendly interface for easy navigation and interaction
  • Integration capability with IoT devices and mobile apps for field use

Advantages of the Project

  • Quick identification of diseases
  • Provides actionable treatment advice
  • User-friendly interface for ease of use
  • Potential for early disease detection and prevention
  • Promotes crop health and agricultural productivity

Future Vision

The project aims to expand into real-time field detection through mobile apps and integration with IoT devices. Future updates will include more disease categories, localization for specific regions, and user-customizable disease databases. We plan to integrate machine learning models that can learn from new datasets continuously, ensuring the model stays up to date with emerging diseases and evolving plant health threats.
